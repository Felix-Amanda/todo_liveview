Descrição

O TodoLiveView é uma aplicação de Lista de Tarefas (Todo List) desenvolvida utilizando:

Elixir: Linguagem funcional e concorrente.

Phoenix: Framework web para Elixir.

LiveView: Permite interfaces web interativas sem JavaScript.

Ecto: Biblioteca de banco de dados para Elixir.

DaisyUI: Extensão do Tailwind CSS para estilização de componentes.

A aplicação permite que o usuário:

Adicione tarefas.

Marque tarefas como concluídas, com destaque visual (texto riscado e cinza).

Exclua tarefas que não são mais necessárias.

Receba mensagens de erro ao tentar adicionar tarefas em branco.

Interaja com uma interface moderna e responsiva.


Pré-requisitos

Antes de executar a aplicação, você precisa instalar algumas ferramentas essenciais no seu computador:

Elixir e Erlang/OTP

Necessários para rodar a aplicação Phoenix.

Garantem o ambiente funcional e concorrente da linguagem.

Node.js

Usado para processar os assets do projeto (CSS e JavaScript).

Banco de dados (SQLite3 ou outro compatível)

Responsável por armazenar as tarefas criadas na aplicação.

Git

Para clonar o repositório e acompanhar o histórico de commits do projeto.


Como Rodar a Aplicação
Passos

Clonar o repositório:
Obtenha uma cópia local do projeto do GitHub, garantindo que você tenha todos os arquivos e histórico de commits.

Instalar dependências:
Instale todas as dependências necessárias do Elixir e Node.js para que a aplicação funcione corretamente.

Configurar o banco de dados:
Crie o banco de dados e aplique as migrações para preparar as tabelas que armazenarão as tarefas.

Iniciar o servidor Phoenix:
Execute o servidor Phoenix para tornar a aplicação acessível localmente.

Abrir a aplicação no navegador:
Acesse o endereço local (normalmente http://localhost:4000) e comece a interagir com a lista de tarefas:

Adicione novas tarefas.

Marque tarefas como concluídas.

Exclua tarefas desnecessárias.

Observe as atualizações em tempo real, sem precisar recarregar a página.
